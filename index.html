📋 完整计算逻辑说明（最终精准版）

一、基础汇率与货值
• 汇率计算：RMB金额 ÷ 汇率 = USD金额；USD金额 × 汇率 = RMB金额。
• 拿货总价 = 拿货单价 × 拿货个数 + 其他杂费。
• 单个产品采购成本 = (拿货总价 + 头程总价 + 总打包费) ÷ 拿货个数。

二、头程物流成本
• 计费重规则：材积重 = (长×宽×高) ÷ 6000，与实重取大值。
• 头程总价 = 总重量(或材积重) × 运输单价。
• 单个平摊头程 = 头程总价 ÷ 总采购数量。

三、尾程物流与仓库 (TTS/CBT/FBT)
• 第三方本土仓：成本 = 库内操作费 + 续件费 + 包材费 + 尾程运费。
  - 出库费 (库内操作费)：自动根据总重量计算 (0-2LB=$0.7, 2-5LB=$1.5, 5-10LB=$2, >10LB=$2)。
  - 包材费：发货1个=0，2-20个=$0.3，>20个需手动填写。
  - 尾程运费：根据选择的 TTS 或 CBT 价卡计算。计费重规则为取实际重量与体积重(长×宽×高(inch)÷166)的较大值。
• FBT仓：成本 = FBT一口价(含操作与配送，内置官方价卡)。
• 运费承担方式：
  - 商家包邮：商家支付全额运费。
  - 客户全担：客户支付全额运费。
  - 共同承担：客户支付固定阶梯运费，剩余由商家支付。

四、FBT特殊政策规则
• 体积重新规：如果一件商品的实际重量≤2磅，且其体积≤332立方英寸，则仅根据实重计算；超过任一阈值，则取Max(实重，体积重)。体积重=长×宽×高÷166(英寸)。
• 默认标准(Standard)：强制商家包邮，商家承担所有FBT费用。
• 退出政策(Opt-Out)：
  - 订单金额(扣券后) ≥ $30：商家包邮。
  - 订单金额(扣券后) < $30：客户强制付$5.99运费，商家获得$0.75/件补贴。

五、平台费用与佣金
• 达人佣金 = (客户最终支付金额 - 客户承担运费) × 佣金比例。
• 平台佣金 = 客户最终支付金额 × 佣金比例。
• 智能营销佣金 = 客户最终支付金额 × 佣金比例。
• 货损/退货 = 客户最终支付金额 × 比例。

六、利润与ROI
• 最终总成本 = 单品采购头程成本 + 尾程物流(商家承担部分) + 仓库/FBT费 + 平台佣金 + 智能营销佣金 + 达人佣金 + 货损。
• 实际收入 = 客户最终支付总额 (含客户运费)。
• 最终利润 = 实际收入 - 最终总成本。
• ROI = 最终利润 > 0 ? (实际收入 ÷ 最终利润) : 无法保本。

七、达人寄样成本
• 本土仓寄样 = (单品采购头程成本 + 库内操作费 + 包材费 + 尾程运费) × 寄样数量。
• FBT仓寄样 = (单品采购头程成本 + FBT一口价费用) × 寄样数量。
• 回本单量 = 寄样总成本 ÷ 单单利润 (向上取整)。

八、需回本销售件数测算
• 国内成本统计 (采购+头程+杂费）总费用 = (拿货总价 + 头程总价 + 总打包费) ÷ 汇率。
• 单件回款贡献 (盈利时)：
  - 本土仓：平台扣(邮+佣+营+达)回款 - 仓库费用
  - FBT仓：平台扣(邮+佣+营+达)回款
• 单件回款贡献 (亏损时)：
  - 本土仓：平台扣(邮+佣+营+达)回款 - 仓库费用 - 单个产品成本
  - FBT仓：平台扣(邮+佣+营+达)回款 - 单个产品成本
• 需回本销量件数(向上取整) = 总费用 ÷ 单件回款贡献。
• 最终纯利润件数 = 拿货个数 - 需回本销量件数。
• 售完最终纯利润件数后利润 = (拿货个数 × 单件回款贡献) - 总费用。
• 现阶段最终盈亏 = (现已售卖件数 × 单件回款贡献) - 总费用。
